## about

- このリポジトリには、生成AI「Claude Sonnet4.5」の出力したコードが多数含まれます。

## initialize, build and run

- CMakeを利用しています。
- SDL3をsubmoduleで配置しています。
    - メモ: git submodule add https://github.com/libsdl-org/SDL.git vendored/SDL
- configure
    - cmake -S . -B build
- build iteration
    - cmake --build build && ./build/main
