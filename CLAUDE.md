# CLAUDE.md

このファイルは、Claude Code (claude.ai/code) がこのリポジトリでコードを扱う際のガイダンスを提供します。


## Claude Codeの作業ルール

- 必ず日本語で応答してください。
- ドキュメントやコメントは日本語で記載してください。
- 作業内容は「./worklog/yyyymmdd_hhss.md」の形でレポートを作成してください。レポートの内容は、「変更した内容の概要を3行程度で」「なぜそのように変更しようと考えたかを3行程度で」記載してください。
- 新しく作業を開始したときは、worklogの直近5件の内容を把握してください。
- コード作成時は、ドキュメントコメントを整備してください。


## プロジェクト概要

- C++20による開発検証サンドボックスです。
- CMakeを利用しています。
- SDL3を利用しています。
- 実装について
    - game.ccはSDL3のコールバックを受けるグルーコードです。
    - game_manager.hもグルーコードですが、window、renderer、joystickなどゲーム実装に必要な状態をC++クラスの形で管理しています。
    - game/test_impl_2.(cc|h)は実装本体の例です。ゲーム実装の本質的な部分の関心を分離しています。
    - draw_helper.hではSDLの描画処理の冗長な関数名を隠蔽し、メソッドチェーン方式で呼びやすくするためのヘルパーです。
